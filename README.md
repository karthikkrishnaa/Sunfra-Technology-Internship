# Sunfra-Technology-Internship
Overview
This repository contains the projects completed during my internship at Sunfra Technologies as an Embedded Systems and IoT Intern.
The projects focus on:
*ESP32 Development
*LoRa Communication
*Modbus RTU Communication
*Industrial Sensor Integration
*WiFi Connectivity
*Process Variable Monitoring
*Water Level Monitoring

## Projects
1. LoRa Communication
Implemented wireless communication between ESP32 nodes using LoRa modules.
Features:
Long-range wireless communication
Data transmission between multiple nodes
Sensor data monitoring

2. Three Node LoRa Network
Developed a multi-node LoRa communication system with one receiver and multiple transmitters.
Features:
Multiple transmitter nodes
Central receiver node
Reliable wireless data exchange

3. Process Variable Monitoring
Implemented process variable acquisition and transmission using ESP32.
Features:
Sensor data collection
Wireless transmission
Real-time monitoring

4. Water Level Monitoring System
Developed a water level monitoring system using ESP32.
Features:
Water level sensing
Real-time monitoring
ESP32-based processing
